Current version and changes

There is a list of the changes that have been made since the first version of H2M (Mar. 18, 1997) in the file h2m/doc/CHANGES. The current version number which should be found in all the m-files is 2.0. Since version 1.3, the code for the multivariate Gaussian case has been pretty stable and changes have mostly concerned the documentation and examples.

The H2M/cnt extension appears only in version 2.0.
