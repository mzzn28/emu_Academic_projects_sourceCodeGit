Downloading H2M

H2M is available as a Unix gz-compressed tape archive both by anonymous FTP at
ftp://tsi.enst.fr/pub/cappe/mfiles/h2m.tgz
and by HTTP at
http://tsi.enst.fr/~cappe/mfiles/h2m.tgz.

A PC zip file h2m.zip is also available in the same directories (I experienced problems unpacking this latter file with older DOS versions of pkunzip but it seems to be OK with more recent tools such as WinZip version 6.2 or latter). The size of both archives is about 1.1 Mb.

As this is mainly due to the signal file, the same directories also contains archives with the subdirectory data excluded named h2m_nodt.tar.gz and h2m_nodt.zip whose size is about 250 kb (you will not be able to run the example ex_sprec of section 2.4 if you download these).

The documentation is available in HTML and pdf formats. The .tgz and .zip archives also contain the documentation in these two formats.
